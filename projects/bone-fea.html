Project

Bone FEA Analysis

Project Overview

Finite Element Analysis project exploring stress distribution, validation, and interpretation of bone-like geometry under load.

Engineering Tools Used

Finite Element AnalysisStress distributionValidationInterpretation

Problem

Biological structures are geometrically complex and materially sensitive, so FEA results need careful setup and interpretation.

Approach

Set up boundary conditions, studied stress distribution patterns, checked model behaviour, and interpreted results against engineering expectations.

Challenges

A model can look precise while still being wrong. The challenge was treating FEA as an engineering tool that requires assumptions, validation, and judgement.

Results

Analysed stress behaviour and built a stronger understanding of how geometry, loads, and constraints influence simulation outputs.

Image Gallery

Key Takeaways

FEA is not just colourful stress plots. It is a disciplined way to ask whether the assumptions, loads, constraints, and results make engineering sense.
