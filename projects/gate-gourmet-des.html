Project

Discrete Event Simulation – Gate Gourmet

Project Overview

Built a DES model to optimise catering operations by testing scheduling and resource allocation strategies before operational changes were made on the floor.

Engineering Tools Used

Discrete Event SimulationWorkflow analysisData interpretationPresentation

Problem

Catering operations include many moving parts: people, stations, timing constraints, transport, and handoffs. The goal was to understand where queues formed and how resource decisions affected throughput.

Approach

Mapped the process, translated key steps into a simulation model, tested scheduling/resource scenarios, and compared how each strategy influenced bottlenecks and workflow efficiency.

Challenges

Operational systems are messy. A useful model needed enough realism to guide decisions while staying simple enough to explain and iterate.

Results

Identified bottlenecks, tested process improvements, and presented opportunities to improve scheduling and resource allocation.

Image Gallery

Key Takeaways

Simulation is most powerful when it turns complex operations into a shared decision-making tool. The work strengthened my ability to connect data, process observation, and practical improvement ideas.
