# Alex Sugiarto Portfolio

Static personal engineering portfolio for GitHub Pages.

## Deploy

1. Push the contents of this `outputs` folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set the source to the repository root or `/docs` if you move these files there.
4. Replace placeholder media in `assets/` with real photos and `Alex_Sugiarto_Resume.pdf`.

## Customization

- Update links in each HTML file for GitHub and LinkedIn.
- Add real project imagery by replacing the inline SVG galleries.
- Put the final resume PDF at `assets/Alex_Sugiarto_Resume.pdf`.
